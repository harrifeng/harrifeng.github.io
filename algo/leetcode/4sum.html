题目

  • 给出一个数组的整数,计算出这个数组中哪四个整数的组合等于给定的target 
Sample Input : S = {1 0 -1 0 -2 2}, target = 0
Sample Output:
(-1,  0, 0, 1)
(-2, -1, 1, 2)
(-2,  0, 0, 2)

思路

  • 4sum是从2sum演化到3sum,然后演化出来的, 核心问题都是如何控制好组合产生的顺序
  • 首先还是要排序
  • 关键在于如何控制好四个变量,因为beg < end这个肯定是要有的,另外有两个变量如何 控制.开始的适合我的思路是设置两个beg两个end,就是把开头和结尾控制住,然后"智能" 的调节.后来发现这是不行的..没有逻辑能够控制住该怎样跳动
  • 争取的方法是array[i]和array[i+1]这两个控制住,然后后面是beg和end,其实把array[i] 和array[i+1]看成是一个变量的话,那就是3sum了

解答

public class Solution {
    public ArrayList<ArrayList<Integer>> fourSum(int[] num, int target) {
        ArrayList<ArrayList<Integer>> ret = new ArrayList<ArrayList<Integer>>();
        if (num.length < 4) {
            return ret;
        }

        Arrays.sort(num);

        for (int i = 0; i < num.length - 3; i++) {
            if (i > 0 && (num[i] == num[i - 1])) {
                continue;
            }
            for (int j = i + 1; j < num.length - 2; j++) {
                if (j > i + 1 && (num[j] == num[j - 1])) {
                    continue;
                }

                int beg = j + 1;
                int end = num.length - 1;
                while (beg < end) {
                    int sum = num[beg] + num[end] + num[i] + num[j];

                    if (sum == target) {
                        ArrayList<Integer> al = new ArrayList<Integer>();
                        al.add(num[i]);
                        al.add(num[j]);
                        al.add(num[beg]);
                        al.add(num[end]);
                        ret.add(al);
                        while (beg < end && num[beg] == num[beg + 1]
                               && num[end] == num[end - 1]) {
                            beg++;
                            end--;
                        }
                        beg++;
                        end--;
                    } else if (sum < target) {
                        beg++;

                    } else {
                        end--;
                    }
                }
            }
        }
        return ret;
    }
}
