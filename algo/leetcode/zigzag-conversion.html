Leetcode 006: Zigzag Conversion

Table of Contents

  • 题目
  • 思路
  • 解答

题目

  • 有如下字母以齿轮状的顺序排列. 
    P   A   H   N
A P L S I I G
Y   I   R
  • 输入是竖着给的,求横着的输出 
    Sample Input: "PAYPALISHIRING", 3
Sample Output: "PAHNAPLSIIGYIR"

思路

  • 注意当列数为1的时候的特殊情况处理, 就算不会做这个题目,也首先要想到边缘情况.
  • 猛一看这题,就是操蛋的无病呻吟题.后来一想,其实是考察你对mod操作的理解.以三个一列 为例, 后面中间那个就是四个一组. 四个的就是六个一组(总是因为锯齿形状,少两个)
  • 然后我们发现,最终的结果肯定是n个不同的字符串(n等于要求分成的行数)
  • 解法就是根据一列几个先mod成不同的小组, 小组里面前几个肯定是小于n的, 他们mod的结 果是x,那么就加入到string[x], 如果mod的结果x大于n(但也肯定小于2*n-2),那么就是 2*n-2-x, 你可以看看是不是这个规律

解答

public class Solution {
    public String convert(String s, int nRows) {
        if (nRows == 1) {
            return s;
        }
        StringBuffer[] sbuf = new StringBuffer[nRows];
        for (int i = 0; i < nRows; i++) {
            sbuf[i] = new StringBuffer();
        }
        int group = nRows * 2 - 2;
        for (int i = 0; i < s.length(); i++) {
            int newi = i % group;
            if (newi < nRows) {
                sbuf[newi].append(s.charAt(i));
            } else {
                sbuf[group - newi].append(s.charAt(i));
            }
        }

        StringBuffer result = new StringBuffer();
        for (int i = 0; i < nRows; i++) {
            result.append(sbuf[i]);
        }
        return result.toString();
    }
}
