Leetcode 022: Merge K Sorted Lists

Table of Contents

  • 题目
  • 思路
  • 解答

题目

  • 给出一组排序好的k个链表, 返回一个排序好的链表 
Sample Input : [{1,3,4}, {2,4}, {6,7}]
Sample Output: {1, 2, 3, 4, 5, 6, 7}

思路

  • 看起来是一个很麻烦的题目, 如果把视野放到每次取K个链表,然后排序.那就非常麻 烦了,而且操作起来非常不实际
  • 这个时候,就要清醒的放弃这个思路. 一般的解法是K个链表合并其实就是K-1次两个 链表合并.
  • 但是如果只想出"一般"的解法,显然枉费了出这个题目的目的. 如果能想到使用"二分" 来解答这个问题. 那么最后的复杂度就降到了logN, 相当理想
  • 二分,就免不了要进行递归.对于递归条件的判断, 会影响程序正确性.
  • 这里的这个二分的应用非常的巧妙,最后是把结果都放在了list的某一个成员里面.也就 是helper函数的return lists.get(left)这一句.

解答

public class Solution {

    public ListNode mergeKLists(ArrayList<ListNode> lists) {
        if (lists == null || lists.size() == 0) {
            return null;
        }
        return helper(lists, 0, lists.size() - 1);
    }

    private ListNode helper(ArrayList<ListNode> lists, int left, int right) {
        if (left < right) {
            int mid = (right + left) / 2;
            return mergeTwoLists(helper(lists, left, mid), helper(lists, mid + 1, right));
        }
        return lists.get(left);
    }

    private ListNode mergeTwoLists(ListNode listA, ListNode listB) {
        ListNode head = new ListNode(-1);
        ListNode tmp = head;
        while (listA != null && listB != null) {
            if (listA.val < listB.val) {
                head.next = listA;
                listA = listA.next;
                head = head.next;
            } else {
                head.next = listB;
                listB = listB.next;
                head = head.next;
            }
        }
        if (listA != null) {
            head.next = listA;
        }
        if (listB != null) {
            head.next = listB;
        }
        return tmp.next;
    }
}
