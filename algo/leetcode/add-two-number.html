Leetcode 004: Add Two Numbers

Table of Contents

  • 题目
  • 思路
  • 解答

题目

  • 给出两个链表,求两个链表的合.链表保存数字是从后向前的 
Sample Input : {2->4->3} + {5->6->4}
Sample Output: {7->0->8}

思路

  • 题目很简单就是模仿加法进位而已
  • 要注意的地方也不少
    • 首先是list1或者list2一个为空的情况下,还是要继续
    • 一个为空的情况下,在循环里面注意null函数是没有next的
    • 两者都空的情况下,进位可能为1,要补全.

解答

public class ListNode {
    int val;
    ListNode next;
    ListNode(int x) {
        val = x;
        next = null;
    }
}

public class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        int rem = 0;
        ListNode head = new ListNode(-1);
        ListNode tmp = head;
        // The logic is EITHER is not null
        while(l1 != null || l2 != null) {
            int v1 = (l1 == null) ? 0 : l1.val;
            int v2 = (l2 == null) ? 0 : l2.val;
            int sum = v1 + v2 + rem;
            head.next = new ListNode(sum % 10);
            head = head.next;
            rem = sum / 10;
            // Be careful, null has no next
            l1 = (l1 == null) ? null : l1.next;
            l2 = (l2 == null) ? null : l2.next;
        }
        if (rem > 0) {
            head.next = new ListNode(1);
        }
        return tmp.next;
    }
}
