Leetcode 014: Longest Common String

Table of Contents

题目

  • 计算一个字符数组里面,最长的公共前缀. 
Sample Input : ["abc", "abcd", "abcccd", "ab"]
Sample Output: "ab"

思路

  • 从多个字符串里面找到公共的前缀,这个问题没有太好的解法,现阶段的解法就是O(n*m), 要有两个循环.n是字符串个数,m是字符串中最短的字符串长度
  • 首要要判断是否数组中至少有一个字符串
  • 拿出数组中的第一个字符串a作为标杆,取a中的第一个char作为target向后面比较,如果 比到底大家都有这个char,那么加入到result字符串,否则直接退出大循环,返回已经存在 的result字符串

解答

public class Solution {
    public String longestCommonPrefix(String[] strs){
        StringBuffer sbf = new StringBuffer("");
        if (strs.length == 0) {
            return sbf.toString();
        }

        for (int i = 0; i < strs[0].length(); i++) {
            char target = strs[0].charAt(i);
            for (int j = 1; j < strs.length; j++) {
                if (strs[j].length() <= i) {
                    return sbf.toString();
                } else if (strs[j].charAt(i) != target) {
                    return sbf.toString();
                }
            }
            sbf.append(target);
        }
        return sbf.toString();
    }
}
