Leetcode 016: 3Sum Closest

Table of Contents

题目

  • 给出一个数组的整数,计算出这个数组中哪三个整数的组合,最接近于给定的target 
Sample Input : S = {-1, 2, 1, -4}, target = 1
Sample Output: 2 ==> (-1 + 2 + 1 = 2).

思路

  • 3sum closest其实是3sum的一个变体
  • 首先还是要排序.
  • 一直记录sum和target的距离,如果遇到距离为0(sum等于target)的情况下,返回sum就可 以了.否则就记录一个最小值,和最小sum.没有距离为0的情况下,就返回最小sum

解答

public class Solution {
    public int threeSumClosest(int[] num, int target) {
        Arrays.sort(num);
        int minV = Integer.MAX_VALUE;
        if (num.length < 3) {
            return minV;
        }
        int minSum = Integer.MAX_VALUE;
        for (int i = 0; i < num.length; i++) {
            if (i > 0 && num[i] == num[i-1]) {
                continue;
            }
            int beg = i + 1;
            int end = num.length - 1;
            while (beg < end) {
                int sum = num[i] + num[beg] + num[end];

                if (sum == target) {
                    return target;
                } else if (sum < target) {
                    beg++;
                } else {
                    end--;
                }
                if (minV > Math.abs(sum - target)) {
                    minV = Math.abs(sum-target);
                    minSum = sum;
                }
            }
        }
        return minSum;
    }
}
