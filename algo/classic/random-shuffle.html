题目

  • 给出一个random函数,可以随机的返回[0.0,1.0)区间的浮点数, 求输出一个不重复的随机 数组, 其取值范围为0到x 
Sample Input : random(), x
Sample Output: {2, 1, 3, 4, x...}

思路

  • 这个问题是著名的Knuth shuffle,也叫random shuffle, 简言之,就是把一个已知的数 列打乱顺序, 常见应用就是我们的音乐播放列表随机播放
  • 问题的解答已经有经典的方式,就是把一个数组从后往前,分成[已经随机][正在随机][未随机] 三个部分, [正在随机]的是一个成员, 然后随机从[未随机]部分抽取一个成员和[正在随机] 进行交换. 这个过程对整个数组进行便利
  • 一个常见的错误就是取和[正在随机]进行交换的成员的时候,不是从[未随机]中选取,而是 从整个数组中选取. 这样看似概率相等,实则有些位置可能被反复选中

证明

  • 某个位置在第i次被选中,那么前i-1次没有选中它的概率为 
    (k-1/k) * (k-2/k-1) * .... (k-i)/(k-i+1)
  • 第i次选中他的概率为 
    1/k-i
  • 所以它前面i-1次没有被选中,而第i次被选中的概率为 
    {(k-1/k) * (k-2/k-1) * .... (k-i)/(k-i+1)} * {(1/k-i)} = 1/k

代码实现(python库源代码实现)

def shuffle(self, x, random=None, int=int):
    """x, random=random.random -> shuffle list x in place; return None.

    Optional arg random is a 0-argument function returning a random
    float in [0.0, 1.0); by default, the standard random.random.
    """

    if random is None:
        random = self.random
        # be careful, we won't take member[0] in xrange(), as if choose
        # from only one member, the possibility is difinite 100%
        # ==> int (random() * (0+1)) must be 0
    for i in reversed(xrange(1, len(x))):
        # pick an element in x[:i+1] with which to exchange x[i]
        j = int(random() * (i+1))
        x[i], x[j] = x[j], x[i]
