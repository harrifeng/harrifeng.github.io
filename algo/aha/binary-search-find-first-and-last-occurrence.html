二分查找法查找数字首次末次出现位置

Table of Contents

  • 题目
  • 思路
  • 解答

题目

  • 给出一个排序好数组,比如{0,0,1,1,1,5,5,6,6,6,6,7,11} 找出其中某个数字第一次 出现和最后一次出现的位置 
Sample Input : {0,0,1,1,1,5,5,6,6,6,6,7,11}, n = 5
Sample Output: {5, 6}
Sample Input : {0,0,1,1,1,5,5,6,6,6,6,7,11}, n = 8
Sample Output: {-1, -1}

思路

  • 看起来是个简单的问题, 二分法查找,然后从查找的地方开始左右延伸,肯定能解决这 个问题, 但是这样一来如果数组里面的数字全部相同,那么在最坏情况下就退化成了一 个O(n)的复杂度
  • 所以这个问题肯定还是得从二分的这个角度改, 二分查找的主干不能改.可以增加一个 变量result(初始化为-1,如果找不到就返回-1). 让result每次记录刚刚找到的暂时 的结果,然后修改查找区间, 再找.再找到后,就丢弃前面的结果啦

解答

#include <iostream>
using namespace std;

int binary_search_first(int arr[], int len, int key) {
    int low = 0;
    int high = len - 1;
    int result = -1;
    while (low <= high) {
        int mid = (low + high) / 2;
        if (key == arr[mid]) {
            result = mid;
            high = mid -1;
        } else if (key < arr[mid]) {
            high = mid - 1;
        } else {
            low = mid + 1;
        }
    }
    return result;
}

int binary_search_last(int arr[], int len, int key) {
    int low = 0;
    int high = len - 1;
    int result = -1;
    while (low <= high) {
        int mid = (low + high) / 2;
        if (arr[mid] == key) {
            result = mid;
            low = mid + 1;
        } else if (arr[mid] > key) {
            high = mid - 1;
        } else {
            low = mid + 1;
        }
    }
    return result;
}

int main(int argc, char *argv[])
{
    int array[] = {0,0,1,2,4,5,5,6,6,6,6,7,11};
    int n = sizeof(array) / sizeof(array[0]);
    cout << "Found 5 from " << binary_search_first(array, n, 5) << " to "
         << binary_search_last(array, n, 5) << endl;

    cout << "Found 8 from " << binary_search_first(array, n, 8) << " to "
         << binary_search_last(array, n, 8) << endl;

    return 0;
}
