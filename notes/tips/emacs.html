Create a file name based on the seconds from Epoch

  • 在Eamcs里面运行bash并且得到结果的方法是, 快捷键是meta + ! 
    M-x shell-command
  • 如果要打印到buffer里面,就先按一个ctrl + u 
    C-u M-x shell-command
  • 而打印从Epoch到现在的"second"的方法是 
    date +%s

Open all some kinds of files under specific folder

  • 有时候想打开一个文件夹下面的所有的名如xxx.html或者legacy*的文件,方法如下:
    1. M-x 输入find-dired
    2. 选定文件夹,比如~/tmp
    3. 弹出Run find (with args), 选择想打开的文件的"类型":
      • 可以是[-name "legacy*"]
      • 也可以是[-type f]来打开所有类型为regular的file,这个参数是和find里面的 -type参数用法一致 
        -type c
       File is of type c:

       b      block (buffered) special

       c      character (unbuffered) special

       d      directory

       p      named pipe (FIFO)

       f      regular file

       l      symbolic link; this is never true if the -L option or
              the -follow option is in effect, unless the symbolic
              link is broken.  If you want to search for symbolic
              links when -L is in effect, use -xtype.

       s      socket

       D      door (Solaris)

Make terminal client use emacs color theme

  • If you want to use Emacs's own color-theme other than SSH client's simple 8 color, you need to :
    1. Set the ssh client terminal type to xterm-256color
    2. Start Emacs and input following to test whether it show 256 colors 
      M-x list-colors-display
